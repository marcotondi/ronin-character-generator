# Ronin Character Generator

## Descrizione
Il progetto "Ronin Character Generator" è un'applicazione Node.js che genera personaggi per giochi di ruolo. Utilizza TypeScript per garantire una tipizzazione corretta e una migliore manutenibilità del codice.

## Struttura del Progetto
Il progetto è organizzato come segue:

```
ronin-character-generator
├── src
│   ├── index.ts          # Punto di ingresso dell'applicazione
│   └── types
│       └── index.ts      # Tipi e interfacce utilizzati nell'applicazione
├── package.json           # Configurazione per npm
├── tsconfig.json          # Configurazione per TypeScript
└── README.md              # Documentazione del progetto
```

## Installazione
Per installare le dipendenze del progetto, eseguire il seguente comando:

```
npm install
```

## Avvio dell'Applicazione
Per avviare l'applicazione in modalità sviluppo, utilizzare il seguente comando:

```
npm run dev
```

Per costruire l'applicazione per la produzione, eseguire:

```
npm run build
```

Per avviare l'applicazione in produzione, utilizzare:

```
npm start
```
