# **App Name**: Ronin Character Generator

## Core Features:

- Stats Display: Display character stats (HP, Virtues, Honor, Ryo).
- Attributes and Abilities Display: Display character attributes and abilities.
- Equipment Display: Display character equipment, weapons and armour.
- Downloadable Sheet: Include a link to download character sheet.

## Style Guidelines:

- Primary color: Saturated Yellow (#FFC857), chosen to represent honor, tradition, and the rising sun in Japanese culture, contrasting against a dark backdrop.
- Background color: Dark gray (#2B2B2B) to give a strong contrast so the content will be easily read.
- Accent color: Orange (#E07A5F), a darker shade of yellow, should be used for interactive elements to draw focus.
- Font: 'Literata', a serif font, to maintain the traditional and literary feel of the setting.
- The app should follow the same structure as the image.
